== Description ==

AI SEO Content Analyzer is a comprehensive SEO tool designed to scan and evaluate your content for potential SEO issues. By examining your content based on predefined parameters, it provides actionable recommendations and insights to improve its SEO optimization. Whether you're looking to increase your content's visibility, address common SEO mistakes, or just ensure that your website is on the right track, this plugin offers a streamlined and user-friendly solution.

== Installation ==

1. Upload the `ai-seo-content-analyzer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, navigate to the specific settings page or menu of the plugin to configure it.

== Frequently Asked Questions ==

= Does this plugin require any other plugins to work? =

Yes, AI SEO Content Analyzer requires an active SEO plugin (Yoast, Rank Math, or All in One SEO Pack) and Classic Editor to function properly.
